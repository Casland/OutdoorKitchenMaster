## Correct Build Order

Your 10-step list was close. Two changes matter:

1. **Plumbing rough-in comes first** (before anything touches the ground) if you're trenching — you cannot dig under a built kitchen.
2. **Electrical rough-in happens *during* framing, before cement board** — boxes and conduit runs go through the open studs. Only *device trim* (outlets, covers) happens late. Doing "electrical" as step 7 after cladding would mean fishing wire through a closed structure.

Final sequence:

```
Phase −1  Plumbing rough-in (drain path decision + stub)
Phase 0   Site prep, conduit relocation, PT base
Phase 1   Steel frame (incl. kamado cradle, backsplash wall, all blocking)
Phase 2   Electrical rough-in (conduit, boxes, wire pulled)
Phase 3   Cement board + waterproofing + flashing + hardware cloth
Phase 4   Cladding (porcelain plank on all visible faces)
Phase 5   Doors & drawers (parallel-track — can start anytime after Phase 1)
Phase 6   Countertop
Phase 7   Backsplash tile
Phase 8   Appliances, plumbing trim, electrical trim, final details
```

# Phase −1 — Plumbing Rough-In (Deferred-Drain Plan)

![Drain stub geometry](diagram-06-drain-stub.svg)

**The decision:** you're going Option C — build the kitchen now, tie the drain into the sewer later, and run a **catch jug under the sink** in the meantime. Two routing candidates for the eventual connection:

**Routing decided:** the first few feet of the actual drain get installed NOW — a short run of pipe seated directly beneath the paver seam, turning up through the sink-bay floor as a capped internal riser. The future yard trench simply meets the capped outside end whenever tie-in day comes; nothing under the island ever needs to be dug again, and no external upstem is needed.

**The now-work:**

1. **Lay the actual drain start — seated HIGH.** (Supersedes the earlier empty-sleeve idea: the routing is decided, so install the real pipe.) The fall budget to the cleanout is known-tight, so elevation is banked deliberately: pull the paver(s) at the crossing, scrape only enough bedding to seat **2" ABS/PVC DWV directly beneath the pavers** — one horizontal-ish run pointed exactly at the future trench exit with a slight fall outward, then a **long-sweep 90 (or two 45s — never a tight vent elbow)** turning up inside the sink bay, then a riser stub **capped 6–8" proud of the pavers** (cut to height at trim-out; tall stubs survive construction). Cap the outside end at/below grade, photograph + measure its location to the fence post. 2" beats 3" here: the trap arm is 1½", and the smaller pipe's invert sits higher — more fall budget. The long sweep means the riser doubles as rodding access for the whole underground run once the trap is pulled. Everything from the trap up (tailpiece → P-trap → trap arm → riser) lives above ground inside the cabinet, accessible forever.
   - **⚠️ Anchor conflict:** this pipe sits at bedding depth, exactly where the Phase 0 **Tapcon anchors** drill. Mark BOTH lines (drain + gas sleeve) on the sill layout before relaying pavers so no anchor, now or ever, lands on either.
   - **Bank height at the trap arm too:** run the trap arm out of the sink as HIGH as the sink allows (18–20" above pavers is realistic with a bar sink at a 36" counter) — free fall budget.
   - **The tie-in math (do on paper before digging):** fall available = trap-arm height above pavers + cleanout invert depth; fall needed = run length × ¼"/ft. Plan at ¼"/ft for 2" line. Comfortable margin → gravity to the cleanout. Tight → **fence-run route** (buys ~1 ft of elevation and nullifies the problem). Negative → **utility drain pump** (Saniflo-class gray-water pump, ~$150–300, under the sink; pressurized small-diameter discharge runs uphill, elevation problem ceases to exist; uses the sink outlet already in the plan).
   - Material note: pick ABS or PVC now and match it (or use transition cement) when the yard trench happens — San Diego is traditionally ABS territory.
2. **Lay the gas sleeve in the same operation — short, straight, seam-crossing only.** A **straight** 18–24" run of 2" PVC Sch 40 (3" for luxurious clearance) in the same paver lift, offset ~12" laterally from the drain, crossing ONLY the footprint seam and emerging inside the island next to the drain riser location — with clear vertical headroom above the exit. Cap both ends, leave a **pull string** inside, sharpie both ends "GAS SLEEVE," photograph the run. Do NOT lay actual gas pipe — pressurized fuel gets installed, pressure-tested, and inspected as one operation on tie-in day.
   - **Why the straight sleeve isn't a problem:** the future line is **CSST** (or PE), which bends by hand — the installer pushes it through the short straight chase and simply bends it upward at the exit. Rigid pipe needed the glued sweep (drain); flexible pipe IS the bend.
   - **Interior routing (tie-in day, or pre-run — see below):** from the riser point, CSST runs to the grill bay **through the centered stud-web knockouts of the back wall** — inside the wall plane, behind everything, no drawer conflict possible. Centered in a 3-5/8" web the line sits 1.5"+ from either face, beyond the reach of 1¼" backer screws, so **no strike plates on the knockout run**; plates (or a verified 1"+ setback) only where the line leaves the centered path — the rise at the stub and the turn into the grill bay. **Grommets/bushings at every knockout regardless** (abrasion protection is a separate rule from strike protection). Installer bonds the CSST per manufacturer instructions. Its own sealed hole through the cabinet floor board, beside the drain riser's.
   - **Pre-run option:** the island's interior stays accessible for life (hardware-cloth back, door/drawer openings), so pre-running buys little — the installer can push CSST through the finished structure in 20 minutes. If pre-running anyway: use **Home-Flex** (the DIY-sold CSST; TracPipe/Gastite require certified installers), confirm the future installer will accept and test homeowner-run line BEFORE buying, cap + tape both ends, and pressure-test with a gauge tee (10 psi, hold an hour) before and after construction. Default: don't.
   - **Chase note:** the gas sleeve is a SEPARATE piece of the same 2" Sch 40 PVC, laid alongside the drain with a few inches of sand between — the drain itself is a live pipe, never a chase. 2" ID (~2.07") passes ¾"–1" CSST easily, fittings included.
   - **Grill fuel decision:** the Coyote ships fuel-specific. Tie-in a year+ out or uncertain → order **LP**, cook on the tank, budget the **Coyote LP→NG conversion kit + orifice/regulator swap** (~$60–100 + an hour of installer time) for tie-in day. Tie-in imminent → order **NG** outright and skip the propane era. Detection flips at conversion: propane sinks (detector low now), natural gas rises (add detection high after).
3. **Lay a third pipe: the universal spare chase.** Same spec as the gas sleeve — straight 2" PVC, capped both ends, pull string, sharpied "SPARE" — in the same lift. Empty conduit under a permanent structure is never wasted: future low-voltage lighting, an added circuit, irrigation control, or a water line. (If the someday-goal is hot water at the sink, the better answer is a **point-of-use heater** under the counter — 2.5-gal mini-tank, ~1440W/12A on the existing cold line + sink-area circuit, hot water in seconds with zero trenching. Remember during Phase 2 that this appliance is probable when weighing circuit loading.) **Trench arrangement across the seam: drain, spare, gas** — a few inches of sand between each, nothing touching, gas outermost so no future dig at a water line ever crosses it.
4. **Build the end bay removable anyway** (cheap insurance): frame that end panel as a screwed-on, cladding-faced access panel rather than a permanently tiled face — same trick as the drawer faces: a tiled cement-board panel through-bolted from inside. Nobody will see the difference, and if the buried run ever needs attention, you have a service door instead of demolition.
5. **Frame around the riser.** The riser comes up through the sink-bay floor, so the cabinet floor's cement board (Phase 3) gets a hole cut around it, sealed with exterior silicone. No side exits, no external upstem — the trap-to-drain connection lives entirely inside the cabinet.
6. **Interim jug setup:** hard-pipe the sink tailpiece + P-trap now so the plumbing above the cabinet floor is done once, and let the trap discharge into a large sealed catch jug (a 5–7 gal carboy/jerry can with a bulkhead fitting is much nicer than an open bucket — no splash, no smell, no mosquitoes). Empty it to a proper drain, not the landscape, if there's anything besides rinse water in it.
7. **Supply:** the existing water line comes up in the wrong spot — route ½" **PEX low and around** the fridge bay (never behind the fridge; built-ins fill the cavity nearly wall-to-wall and you don't want supply against the compressor/exhaust path).

**Straight talk for later, so it's on the record:** in San Diego, sink fixture waste is supposed to go to the sanitary sewer, the underground portion normally requires an inspection **before it's covered**, and a sewered outdoor sink needs a roof or a permanently attached cover. None of that blocks the build today — the jug is a legitimate bridge — but when you do the tie-in, doing it permitted protects you at resale and the homeowner no-plan permit is cheap relative to the trenching labor you're doing anyway. Confirm the path with [San Diego DSD](https://www.sandiego.gov/development-services).

# Phase 0 — Site Prep, Conduit Fix, and the Level Base

### Step 0.1 — Verify the paver base
Walk the footprint. Look for rocking pavers or ponding low spots. A loaded island runs 1,500–2,500 lb; if pavers are on proper compacted road base + sand, you're fine. Fix any rockers now.

### Step 0.2 — Electrical conduit relocation (your step 1)

![Conduit offset at the sill](diagram-02-conduit-offset.svg)
The existing stubs sit tight to the fence post and would collide with the back plane.

1. **Kill both circuits at the panel. Verify dead.**
2. Demo the existing boxes; cut each PVC stub down, leaving a coupling's worth above the pavers.
3. **Confirm what the second pipe in the double stub is** (spare circuit? low voltage? gas?) before demo — it changes the plan if it's not a spare.
4. Glue on couplings and **PVC offset fittings right at the base, below track level** (~$2 each; or heat-gun-bend your own). The jog kicks the conduit off the fence line so it rises *inside* a stud bay. Never notch the back cement board at grade.
5. Lay out the run so each stub lands **between studs**, not under track. If one lands dead-on the track line, notch the track flange around it — not the pipe.
6. Cap the open conduit ends until Phase 2.

### Step 0.3 — Lay out the footprint
- Snap chalk lines on the pavers for both legs of the L, holding the whole island **1.5–2" off the vinyl fence** (you already have scorch marks on that fence — vinyl deforms ~160°F; the gap also makes your back vents functional and gives the conduit room).
- Mark the sink chase, conduit penetrations, and the grill/kamado/fridge bay centerlines on the ground.

### Step 0.4 — Build the PT base (your step 2)
Locked: PT 2×4 shimmed directly on the pavers.

![Base and finish stack cross-section](diagram-01-base-finish-stack.svg)

*Read the section above before cutting anything: the track's front flange sits flush with the PT front face, the stud web spans the full track interior, and the finish stack (½" board + thinset + plank ≈ 1") drops past the sill to a weep screed ~½" above the pavers — no raw wood visible at grade. If you'd rather have a recessed toe-kick at any bay, hold that bay's board/cladding up at a horizontal trim line instead and paint the recess dark; the default here is the monolithic face to grade shown in your renders.*

1. **Find the high point** of the pavers inside the footprint with the laser. That's your datum — everywhere else you pack *up*; never grind down.
2. **Lay the drain start** (Phase −1): the 2" pipe + riser goes in at the sink-bay crossing before any sill goes over it — and its marked line is a no-drill zone for every anchor in step 6.
3. Cut PT 2×4 (above-ground UC3A) sill runs for every wall line, including the backsplash wall and the kamado cradle.
4. **Gap or drill the sill** where the conduit stubs rise (drilling a hole ~¼" oversize mid-board is stronger than an open notch; PVC needs the slack for thermal movement).
5. Set each run on the chalk lines. Shim to the laser at every bearing point (cedar shims were on the buy list; composite shims are the more rot-proof upgrade if you want it). Check level along and across every run, then leg-to-leg across the whole L.
6. Bed in **PL Premium** and anchor to the pavers with Tapcons/paver anchors every few feet. The sill is your shear connection to earth.
7. **Isolation is non-negotiable:** roll **closed-cell sill seal foam** over the top of every PT run before any steel touches it — modern ACQ/copper-azole PT corrodes galvanized steel. Any screw touching PT is **hot-dip G185 or stainless**, never plain zinc.

**Check before moving on:** laser across every run and diagonal — you want the whole sill plane within ~1/16". Every error here telegraphs to the countertop.

# Phase 1 — Steel Frame (your step 3)

### Materials (from the phased-budget chat)
- ~**50 sticks** 3-5/8" × 10' 20-ga **EQ** studs (label reads -18/-19/-20/-23 mil; avoid -14/-16). The 10-footers yield **three ~33" studs per stick** — that's why you chose them.
- ~**14 sticks** 3-5/8" track (U-channel — no return lips; don't mix up with studs).
- #8 × ½" wafer-head self-drilling TEK screws (500 box) — stud↔track. ~4 per stud.
- #10 × ¾" hex-head TEK screws (100 box) — track→PT sill (these bite the wood better).
- Cold-galvanizing spray, 2 cans — **every cut edge gets sprayed.** Non-negotiable outdoors.
- Sources: J&B (El Cajon) or Westside (Sorrento Valley) supply yards beat HD on studs; buy TEK screws online/Fastenal, not HD.

### Step 1.1 — Cut the studs
- **Stud length is locked: 32-13/16"** (34" − 1-3/16" for the 3cm granite). Cut a test stud, dry-stack the full sandwich (PT + track + stud + track + ½" deck board) and tape-measure to 34-13/16" before you gang-cut fifty of them.
- Score-and-snap: snip both flanges, nick the web, bend and snap. Or the miter saw with a **carbide metal-cutting blade** (Diablo Steel Demon type — cuts cold, no sparks, clean miters). Never an abrasive wheel.
- Deburr every cut; **spray every cut edge** with cold-galv. Wear cut gloves — sheared steel is a razor.

### Step 1.2 — Assemble wall panels flat, tilt up

![Track splice and outside corner](diagram-03-splice-and-corner.svg)
- Build each wall section flat on the patio, square it with a framing square, then tilt into place onto the sill track. Far easier than building vertically.
- **Sheath backs before tilt-up:** a panel lying flat has an accessible back face; a standing panel against the fence never does again. The grill-bay section gets its back cement board attached NOW, on the ground (see Phase 3 zoning). Any storage bay you want dust-tight gets the same treatment — decide per panel before it stands. (The curved corner is framed in place, so staple its hardware cloth progressively as you frame, while your arm still fits.)
- Studs at 16" o.c. standard; **~6" o.c. on the kamado arc/corner facets** if you kept any curved/faceted geometry.
- **Clamp before you drive** — locking C-clamp pliers (own 4–6) hold the layers so the metal can't walk. Drywall screw gun with adjustable nose > impact for TEKs; a stripped hole is a dead joint.
- Corners: two mitered pieces spliced with a nested track sleeve, screwed through both flanges + web — stronger than a single bent web. Splice long runs with an 8–12" nested sleeve.

### Step 1.3 — Fasten to the base
- **Weep the tracks first, on the sawhorses:** drill ¼" holes at the very bottom of the BACK flange, right at the web corner, 1–2 per stud bay, every track run. The U-channel is otherwise a gutter with no outlet — cavity water (wind-blown rain through the back cloth, or a someday plumbing mishap) pools in it and standing water is what eats galvanizing. Because the track's back flange slightly overhangs the PT's rear edge (front flange is the flush one), these weeps drain sideways past the PT face into the fence air gap and onto open paver. Deburr + cold-galv every hole. Same philosophy as the cabinet-floor weeps and the weep screed: never give water a place to sit.
- Bottom track over the sill seal, onto the PT, with the #10 × ¾" hex TEKs (G185/stainless where they touch PT).
- Plumb every wall, brace temporarily, then tie the legs of the L together into one rigid unit.

### Step 1.4 — The kamado cradle (build this heavy) — Big Joe, locked

![Kamado cradle load path](diagram-05-kamado-cradle.svg)
A Big Joe is the heaviest single object in the build — the standalone Series III grill head runs roughly **28.2"W and on the order of 400+ lb** of ceramic concentrated on a small footprint.

- The load must travel **straight down in compression**: posts directly under where the ceramic base contacts (4 bearing points for a Big Joe), running uninterrupted to the base track. Never carry it on a horizontal stud spanning an open bay.
- Use your **doubled-rail grid** here (you already modeled this — good). Web-vertical orientation or boxed track for any horizontal members, never flat track on its weak axis.
- Deck the shelf so the base bears on a continuous surface, not thin flanges (cement board deck comes in Phase 3; mortar-bed the feet zone).
- **Kamado Joe requires ceramic feet in built-in installs** — Big Joe takes **4** (2 front, 2 back), spaced so it doesn't rock when the dome opens. They create the under-grill airflow that dissipates heat; don't set the ceramic base flat on the deck. ([kamadojoe.com parts](https://www.kamadojoe.com/collections/parts))
- Dimension the cutout from the **A–R built-in chart**: [Ceramic Grill Store — Kamado Joe Built-In Dimensions & Cutout Guide](https://ceramicgrillstore.com/products/kamado-joe-built-in-dimensions-cutout-guide). Two rules from it worth engraving: **don't build fixed counter in front of the grill** (the front latch needs access), and the shelf surface sits a few inches **below** counter so the band/latch clears.
- Verify the dome's rear swing clears the backsplash/curved wall before you commit the shelf depth.
- Controller: order the **SMOBOT Kamado Joe version** ([smobot.com/products/kamado-joe](https://smobot.com/products/kamado-joe), ~$349) — the standard BGE daisy-wheel kit does **not** fit the cast-aluminum Kontrol Tower. It's micro-USB powered, so the outlet at the cradle serves it and the Looftlighter.

### Step 1.4b — The curved corner wall (curve is alive)

![Curved wall layer buildup](diagram-04-curved-wall.svg)
The studs do the curving; everything above them just follows.

1. **Curved track:** snip both flanges of a length of track every 1–2" (tighter radius = tighter snips), leave the web intact, and bend it smoothly to your arc. Cut one for the base and a matching one for the top; trace the arc from your SketchUp template onto plywood/cardboard first so both tracks match exactly.
2. The PT sill under it: cut short PT segments to chord the arc (the sill doesn't need to be a perfect curve — it just needs bearing under the track's snipped web every few inches), or band-saw a curved sill from wider PT stock if you want it clean.
3. **Studs at ~6" on center around the arc.** At that spacing the facets disappear under the finish. Plumb each one; the two tracks hold the geometry.
4. **Pre-install the gas chase through the corner** while the framing is bare: 1¼–1½" PVC conduit routed through the corner's stud knockouts (gentle sweeps or straight chords), both ends emerging in the open bays flanking the corner. Cap, pull string, label "GAS CHASE." Once the cradle carries the Big Joe against the fence, this corner is blind forever — this $8 of pipe is the only way a future line crosses it in twenty minutes instead of never.

### Step 1.5 — Backsplash wall
- Frame the thin backsplash wall per the stud-orientation plan we worked out (studs rotated for minimal thickness).
- Remember it can't be anchored into pavers against wind load — its stability comes from being tied hard into the L and the mass of the island. Bolt it through into the counter framing with blocking at the bolt lines.

### Step 1.6 — Openings and blocking (do-now-or-regret-it list)
Frame every negative to the **manufacturer's cutout spec, not your framing instinct**:
- Grill: ⚠️ **decide SL30 vs SL36 before framing this bay.** SL30: 25½"W × 20½"D × 10¼"H · SL36: 32½"W × 20½"D × 10¼"H — plus ~1" margin on width/depth for cross-brand swap insurance. Depth is where brands vary most. (Everything else in the frame is identical either way; only this one bay and its flanking stud positions wait on the decision.)
- Fridge: the Velieta 24" class — pull the exact cutout from its manual once it arrives (frame that bay last if needed; these budget units share ~23.5"W × ~23"D × ~34"H footprints but verify). Frame a **solid, dead-level floor shelf** for it. Honesty note: at ~$350 it's a verified front-vent built-in, but it's not a Blaze-tier outdoor unit — the lined cavity gives it a sheltered life, and if it dies in 6–8 years, the standard 24" bay means any replacement drops in.
- Sink cutout per sink template.
- **Horizontal blocking at counter height** under the stone, grill flange, and sink.
- **Hinge backing at every door opening:** bolt aluminum angle (or boxed heavier-gauge stud) across the hinge-side stud *now* — a thin stud web alone will not hold a piano hinge for 20 years, and this is a miserable retrofit.
- **Shelf blocking/ledgers** at your chosen shelf heights in every cabinet bay — cement board is skin, not structure.
- Route paths: confirm the conduit bays, the PEX route low around the fridge, the drain chase.

**Check before moving on:** laser across the entire top track — dead level, both legs, corner to corner. Rack-test every bay. This is the last cheap moment to fix anything.

# Phase 2 — Electrical Rough-In (your step 7, moved here)

You're doing this yourself and you've got two circuits at the stubs.

### The plan we settled
- **Circuit 1 (20A, dedicated):** fridge. Refrigeration hates sharing with a tripping GFCI.
- **Circuit 2 (20A):** grill outlet (the Coyote SL needs 120V for lights/ignition), kamado outlet (Looftlighter + fan controller — note some kamado electronics are GFCI-sensitive; a clean dedicated-ish path helps), sink-area convenience outlet.
- **GFCI protection on everything** — at the head of each circuit or breaker-level.

### Steps
1. From the offset fittings at the base, extend **PVC conduit** up the stud bays and laterally **through the stud knockouts** (bushings/grommets on any cut steel edge). Conduit runs the whole way — the island interior is a damp/wet location; THWN-2 in conduit, no Romex, no bare UF.
2. Set **weatherproof boxes with in-use covers**:
   - Grill outlet: low (8–12") inside the adjacent cavity or on the back face under the counter — cords out of sight, covers stay closed.
   - Fridge outlet: **in the adjacent cabinet**, not behind the fridge (you can't reach it and the cord gets pinched). Drill a grommeted cord pass-through, seal it (steel wool + sealant — pest highway otherwise).
   - Sink convenience: high on the cabinet sidewall, away from trap drips.
   - Kamado: near the cradle.
3. Keep splices in an **accessible** j-box (the fridge cavity works as a hub — a removable panel counts as accessible; buried behind tile doesn't).
4. Pull THWN-2, land nothing yet — leave tails coiled in boxes. Devices and covers go in at Phase 8 (after tile, so nothing gets thinset in it).
5. Any electrical near the propane bay mounts **high**, not low where gas would pool.

# Phase 3 — Cement Board, Waterproofing, Flashing (your step 4)

### Materials
- ½" cement board (Durock/Hardie), ~**26 sheets** of 3×5 estimated for faces + counter deck + backsplash (recheck against final plan — the open back removed ~100 sq ft from earlier estimates, so re-count before buying; buy 2–3 extra).
- **Power Pro #9 × 1¼" ceramic-coated wafer-head backer screws** — the 925-count box covers the whole project. (Yes, 1¼" is right: ½" board + ¾" past it to thread the steel.)
- Alkali-resistant mesh tape + thinset for seams.
- RedGard or AquaDefense (~2 gal), Z-flashing, weep screed, ¼" galvanized hardware cloth.
- Rockwool ComfortBatt R-15 (1 pack does grill bay + fridge bay + sink back).
- **Respirator for every cut. Non-negotiable.** Score-and-snap where you can; diamond blade outside for the rest.

### Steps
1. **Interior cavity liners first** where access closes later: line the fridge cavity on all five faces (offcuts are fine), build its level floor shelf, and give every cabinet floor a slight pitch (~¼") to a **weep hole** at the low point — no bathtubs. Under-sink bay gets the waterproof-pan treatment on its floor.
2. **Plan and cut every penetration before hanging** — conduit, PEX, drain chase, vents. Sleeve and seal with exterior silicone. Retrofitting holes through finished board is miserable.
3. Hang the **face sheets**: screws every 8" into studs, edges landing on framing, 1/8" gaps between sheets.
4. **Grill bay — a fully boarded compartment, single layer throughout:** back (pre-attached at tilt-up) plus the bay-facing side of BOTH flanking partitions. Not a fire-rating issue (the steel assembly is already non-combustible, ASTM E136) — the side boards do three other jobs: they stop firebox heat and grease vapor migrating into adjacent storage (where the door backers are polyiso foam), they make the bay a defined compartment so the manual's ventilation-opening math means something (LP compartments have explicit vent requirements — low openings especially, propane sinks; size the toe-kick louvers to the manual's square-inch numbers), and they stiffen the partitions carrying the counter at the cutout. Finish level: bare board, seams taped, no membrane, no tile — nobody sees inside a grill bay. Bottom stays open to the pavers for the tank; front per the manual's access + vent layout. **The manual is law:** cutout dims, vent areas, clearance-to-combustibles (the vinyl fence counts — boarded back + 2" gap covers it, verify against the manual's rear number; a cement-board standoff shield on the fence is the cheap fix if tight). Optional retrofit: Rockwool batt into the open stud cavity if the adjacent bay ever runs warm — buy it then, not now.
5. **Counter deck:** ½" cement board screwed down across the top track — it's the counter's subfloor. Everything flush; any blocking sitting ¼" low reintroduces voids under stone.
6. **Back of the island:** **¼" galvanized hardware cloth across the entire back EXCEPT the grill bay.** Cloth staples/screws on from the front through the bays — free ventilation, pest exclusion, a permanent drying lung for the cavity, and physically incapable of trapping water. The **grill bay back gets solid board, pre-attached while its panel lies flat in Phase 1** (no radiant line-of-sight through mesh to the already-scorched vinyl fence). The kamado corner stays cloth: a kamado is a thermos — its ceramic keeps the exterior cool — and the boarded, porcelain-faced backsplash wall already stands between body and fence.
   The boarded back gets seams taped but NO membrane — it is the drying side by design.
7. Tape + thinset all seams and corners (alkali-resistant mesh).
8. **Waterproofing where it earns its keep:** roll RedGard/AquaDefense on the counter deck (top + back edge) and all faces getting cladding. Two coats, opposing directions.
9. **Flashing:** weep screed at the base perimeter; **Z-flashing across the backsplash top plate**, sloped toward the fence, front edge caulked.
10. **Vents:** louvered stainless vents low in the toe kick under the grill and at the island ends; back vents work because of the fence air gap. Add the **LP leak detector** low in the propane bay.
11. If the shell will sit before cladding: fine in rain, but the membrane doesn't love months of direct UV — aim to clad within **2–4 months**, and tape/cap the open counter-deck edges in the meantime.

### Step 3.5 — The curved wall: faceted board → true radius
You're right that you won't bend ½" board around that arc, and you don't need to. The system is: **faceted cement board for the fireproofing, metal lath + mortar for the geometry.** The board is the non-combustible substrate; the lath-and-mortar layer is what actually creates the smooth curve.

1. **Skin the arc in ¼" cement board, cut into vertical strips** — one strip per stud bay (your studs are at ~6" o.c. on the arc, so strips run ~6" wide). Each strip is dead flat, spans stud to stud, zero bending stress, impossible to crack. You're building a polygon that *reads* as a curve. Two screws per strip per stud with the same Power Pro backer screws.
   - *If you want fewer seams:* two alternates from our earlier session — kerf the back of a wider piece every ~1" with a grinder + diamond blade and wrap it, or laminate two ¼" sheets (each bends easily; bonded together with thinset + screws they behave like ½"). The vertical strips are still the most foolproof.
2. Pack the facet joints with thinset and tape everything with alkali-resistant mesh.
3. **Staple/screw galvanized expanded-metal lath over the whole arc**, wrapping it continuously — the lath bends to any radius effortlessly, and it's your true curve former. Overlap laps by 2"+, fasten into every stud. This is your "bendable non-combustible material" — it's exactly what pros use for curved masonry surfaces; a flexible cement board that does this doesn't really exist.
4. **Trowel a mortar/stucco base coat over the lath** (a polymer-modified base coat or sanded thinset works), screeding to the radius. Make a simple **curved screed template** — a piece of plywood band-sawed to the arc from your SketchUp profile — and drag it around the wall to shave the mortar to a perfect radius. Two thin passes beat one thick one.
5. Once cured, finish however the corner wants: a smooth stucco finish coat as a deliberate accent zone (my lean — a smooth curved band framing the red kamado looks intentional next to the planked flats), or narrow mosaic/picket tile which follows a radius happily. Full-width porcelain planks won't wrap it — plan the plank field to *stop* at the curve with a metal trim line, and let the curve be its own material.

Everything in this stack is non-combustible — cement board, steel lath, mortar — so the kamado's radiant heat is a non-issue.

# Phase 4 — Cladding (your step 5)

**The decision:** wood-look **porcelain plank**, run **vertically** for the reeded look in your renders. One material discipline across the whole build (plank faces + picket backsplash + stone/porcelain top = one wet saw, one thinset, one grout strategy).

### Buying specs
- **Rectified** plank (tight, wood-like joints), light driftwood tone per the renders.
- Leveling clip system (Tuscan/Raimondi/MLT) — non-negotiable on planks.
- Polymer-modified exterior thinset; **epoxy grout** color-matched dead-on to the plank so joints read as reveals.
- **Schluter-type metal edge trim** (Quadec/Rondec) for outside corners — never miter porcelain corners; trim is faster, tougher, forgiving. ([schluter.com](https://www.schluter.com))
- Buy one extra box as attic stock — dye lots disappear.
- Saw check: 48" planks fight a small tabletop wet saw — plan on a rail/bridge saw rental or score-and-snap + grinder for notches.

### Steps
1. **Dry-lay the entire face on the ground first.** Shuffle planks for grain/color variation, reject the ugly ones, decide where door openings land so **door edges fall on grout joints**. This is where the forgiveness lives — audition everything before anything is permanent.
2. Snap plumb/level layout lines. Vertical planks, **offset ≤33% — never 50% brick pattern** (plank bow guarantees lippage at 50%; manufacturers warn against it).
3. **Back-butter every plank** + combed thinset on the wall — full coverage.
4. Set with leveling clips; check plane constantly with a straightedge.
5. Edge trim bedded in the same thinset pass as the adjoining tile (the perforated leg embeds in the bed — trim and tile go on together, not sequentially).
6. Cure, then epoxy grout. Clean haze aggressively as you go — epoxy haze is forever.
7. **Movement joints:** color-matched silicone (not grout) at every change of plane and where cladding meets counter/appliances/trim.

# Phase 5 — Doors & Drawers (your step 6)

This runs as a **parallel bench-work track** — build them any time after framing; hang them after cladding.

### Doors (no welding)

![Door construction layers](diagram-07-door-layers.svg)
1. **Frame:** 1"×1" aluminum angle or thin-wall 1" square tube, cut on the miter saw with a **non-ferrous blade**. Corners joined with aluminum gusset plates/corner brackets — stainless machine screws + nylock nuts **plus** structural adhesive (3M 5200 / DP420). Add a center rail on anything over ~24". Size = opening minus an even 1/8–3/16" reveal.
2. **Backer:** **GoBoard/Wedi foam board**, not cement board — waterproof, bonds porcelain equally well, a fraction of the weight (door weight kills hinges). Bond with thinset/adhesive **and** screw with stainless self-tappers; the bond makes the torsion box.
3. Waterproof the backer edges/seams.
4. **Hinge prep before tiling.** You want the **European-style opening** — that points at **concealed hinges made for aluminum-frame cabinet doors** (clamp-on type, no 35mm cup bore, 3-way adjustable — search "aluminum frame door concealed hinge" / glass-cabinet hardware category). Piano hinge remains the sag-proof fallback if the clamp-on options don't suit your frame profile. Final pick happens here at Phase 5 — but either way the **cabinet-side hinge backing was installed back in Phase 1**, so nothing upstream waits on it. Whichever hinge: machine screws + lock nuts into the frame, never self-tappers, for the connection carrying the door's whole life.
5. Tile the face with the same planks — **edges landing on grout joints** so the reveal hides in a reveal; metal edge trim around the perimeter; epoxy grout.
6. **Diamond-drill handle holes** through the tile into the frame, sleeve with silicone, through-bolt stainless handles.
7. Hang, and dial the reveals even. Outdoor magnetic/roller catch.

### Drawers — DIY riveted aluminum (locked, custom sizes)
- Boxes: **.050–.063" aluminum sheet or 3–4mm ACP/Dibond panels**, every corner joined with 1"×1" aluminum angle + **stainless pop rivets** ($25 rivet gun). The angle is the frame; the panel is the skin.
- Run **aluminum flat bar/angle along each side at slide height** — solid bite for the slide screws and it stiffens the wall in one move.
- **Perforated aluminum bottom** so any water drains instead of pooling; **U-channel edge cap** on the top rim (no razor edges, looks finished).
- Boxes land under 10 lb empty → standard **stainless full-extension slides** carry them easily; mount slides to the flat-bar rails and to blocking/angle on the cabinet side.
- Cut list tip: size each box to (opening − slide clearance per the slide spec, usually ½" per side) and build the tallest drawer first as your pattern.
- The **face** stays easy: tiled ½" cement-board panel (weight is fine here — it rides the slides flat) **through-bolted from inside the box with slotted holes** so you can nudge alignment. No hinges, no frame needed.
- Stainless (not aluminum) stays the rule for anything in the firebox surround zone.

### Cabinet interiors
- Light-colored finish (elastomeric/masonry paint over the cement board) so you can see into them.
- Shelves on the blocking you set in Phase 1: stainless wire or marine HDPE are the picks — they drain.

# Phase 6 — Countertop (your step 8)

**Locked: light-colored granite, 3cm (1¼" nominal), leathered finish.** The color rule is non-negotiable outdoors — Alaska White / Kashmir White / Colonial White / Bianco Antico class stones stay usable in the sun; anything dark (Black Pearl, Absolute Black) hits 150–170°F and is off the table. Remnant-shop it: budget remnants run $10–18/sq ft material + $600–1,000 fabrication for your ~40 sq ft, landing $1,000–1,800 all-in. The L needs **two pieces with a seam at the corner** — keep the seam away from the sink, and match thickness across both pieces. Sealing habit: twice a year outdoors, $25 bottle, 20 minutes.

### Steps
1. Template **after** cladding (the finished face is the reference), 31" deep, front overhang over the plank face, rear overhang carrying the backsplash.
2. Give the fabricator: sink model + template, grill cutout (SL30: 25½ × 20½ / SL36: 32½ × 20½ — locked by then), kamado opening per the Big Joe built-in chart, seam location. Cutouts need meat around them — this constrains which remnants work; match 3cm across both pieces.
3. Before install: confirm the deck is level and waterproofed, blocking is flush, and the kamado throat band has its reinforced continuous bearing (the "little shelf" of blocking we detailed — flush to deck, clip-angled into flanking studs, stone fully mortar-bedded over it so it never spans air).
4. Set in silicone beds; seam epoxied and color-matched by the fabricator.
5. **Counter-to-backsplash joint gets silicone, never grout** — it's a movement joint.

# Phase 7 — Backsplash Tile (your step 9)

![Backsplash zones at 18 inches](diagram-08-backsplash-zones.svg)

The framing, board, flashing, and waterproofing are already done — this phase is tile only, **after** the counter so the tile sits on/over the finished surface with a silicone joint.

**Locked: 18" band height, no stainless panel.** The grill heat zone gets **smooth, large-format porcelain** (24"+ faces, minimal grout lines — grout directly behind a grill is a grease trap, so big smooth tile is the wipeable substitute for the stainless we ruled out). **Tile pattern/color stays late-binding** — nothing upstream depends on it; just have it chosen before the counter is templated so the fabricator knows the splash sits on the rear overhang.

### The zone split at 18" (~18 LF of back wall)
| Zone | Width | Material | Quantity |
|---|---|---|---|
| Behind grill | ~4' | Smooth 24"+ light porcelain | ~12 sq ft |
| Kamado corner | ~3' | Smooth porcelain — or bend the rule and frame the egg with the feature tile (soot on grout over time) | ~5 sq ft |
| Decorative (sink wall + runs) | ~11' | Feature tile (your pick of the four comps — Sea Glass Sage was my argument), +20% waste (picket shapes waste more) | ~20 sq ft |

### Rules
- Porcelain or glass only — never natural stone behind a cook surface.
- **Epoxy grout**, one shade *darker* than instinct (pale grout behind a grill grays in a season anyway).
- Smooth-tile-to-feature-tile transitions: metal edge trim + silicone at any change of plane or material, not a grouted butt joint.
- Buy one extra box of the picket as attic stock.

# Phase 8 — Appliances, Trim-Out, Final Details (your step 10)

### Electrical trim
- Install GFCI devices/receptacles, in-use covers, label the circuits, test every GFCI.

### Plumbing trim
- Set the sink in silicone per template; faucet (cold-only), supply hookup to the PEX run, trap + drain connection to the stub, AAV under the counter if that's your vent plan. Final inspection if applicable. Install/verify the **sink cover** (the sewer-connection rain rule).

### Fridge
- Slide into its lined cavity onto the level shelf; **front grille completely unobstructed** — no counter lip, face-frame, or trim creeping over it. Cord through the grommeted pass-through to the outlet in the adjacent bay; seal the pass-through.

### Grill
- Drop into the cutout per the Coyote manual — flange bears on the counter, sealed per instructions. Connect propane (tank on the pavers in the open bay), **leak-test every fitting with soapy water**. Plug in the 120V. Confirm the manual's clearances and (if specified) the insulated jacket.

### Kamado (Big Joe)
- Assemble in place — it does not fit through openings after the fact, and the ceramic body is a two-strong-people lift minimum (rent/borrow a shoulder dolly or appliance dolly and extra hands).
- Set it on the **4 ceramic feet** (2 front, 2 back, spaced so it can't rock when the dome swings) on the cradle deck — the feet create the required under-grill airflow; never flat on the deck.
- Verify dome swing clears the curved wall/backsplash and the front latch is reachable below counter level.
- Swap the Kontrol Tower damper hardware for the **SMOBOT Kamado Joe kit** per its instructions, route its micro-USB to the cradle outlet, and stage the Looftlighter there too.

### Final punch list
- [ ] Silicone every change-of-plane joint (counter↔splash, counter↔grill flange, cladding↔trim, tile↔stainless)
- [ ] Weep holes clear in every cabinet floor
- [ ] Vents unobstructed; LP detector powered/tested
- [ ] Seal granite (if granite) — and calendar it twice a year
- [ ] **Re-level check after a few weeks loaded + one good rain** — early settlement happens once; catch it, shim/adjust, done for the long haul

# ⚠️ Remaining Open Decisions

Everything else is locked. Only four forks left, none of which block starting the build tomorrow:

1. **Grill size — Coyote SL 30" vs 36".** *Deadline: before framing the grill bay* (Phase 1, Step 1.6). Everything else in the frame is identical; only that bay's width and its two flanking studs wait on it. If you're stuck: re-run the cardboard mockup at 25½" vs 32½" cutout widths on the actual sill once it's down — seeing it against the real base decides it in five minutes.
2. **Backsplash tile.** *Deadline: before the counter is templated* (Phase 6) so the rear-overhang detail is confirmed — and practically before you order Phase 4 tile if you want one combined tile order. Sea Glass Sage remains my argument next to the red Big Joe.
3. **Door hinges.** *Deadline: Phase 5, before tiling the door faces.* You want the European-look opening → shortlist clamp-on concealed hinges for aluminum-frame doors; piano hinge stays the fallback. The Phase 1 hinge backing supports either, so nothing upstream waits.
4. **Sewer tie-in timing + permit.** No build deadline — the jug bridges it. The drain start is already laid under the island with a capped end waiting in the yard, so the future trench never touches the built kitchen. When you do it, permitted is cheap insurance relative to the trench labor you're paying for anyway.
